# parameterItem

With the list of parameter properties and the configuration file name

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `props` | parameterProp[] | No | The list of parameters properties |
| `additionalProps` | parameterProp[] | No | The additional properties of the parameter |
| `fileName` | string | No | The name of the configuration file |
| `specName` | string | No | The name of the config spec |
| `rawContent` | string | No | The raw content of the configuration file, return only when raw is true in the query. |
