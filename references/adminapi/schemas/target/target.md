# target

Identifies one side of a scheduling rule match, including component and role.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `component` | string | No | Component name to match. Use `*` to match all components for the selected engine mode.
 |
| `role` | [target_role](target-role.md) | No |  |
