# project

Project with detail information.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | The name of the project. |
| `createdAt` | string (date-time) | Yes | The time the project was created. |
| `envID` | string | Yes | The environment ID of the project. |
| `description` | string | No | The description of the project. |
