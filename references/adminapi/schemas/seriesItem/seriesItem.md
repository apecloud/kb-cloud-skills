# seriesItem

A data point in a series for meta data, including a count and a timestamp.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `value` | number (double) | Yes | The value of the series data point. |
| `timestamp` | integer (int64) | Yes | The timestamp representing the time of the data point, unit is seconds. |
