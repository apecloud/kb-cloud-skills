# organizationItem

Organization items.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | The name of the organization. |
| `displayName` | string | Yes | The display name of the organization. |
| `creator` | string | No | The display name of the creator of the organization. |
| `creatorID` | string | No | The ID of the creator of the organization |
| `clusterTotal` | integer | Yes | The total number of clusters in the organization. |
| `createdAt` | string (date-time) | Yes | The time the organization was created. |
| `description` | string | No | The description of the organization. |
