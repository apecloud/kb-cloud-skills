# imageRegistry

Represents an image registry.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | The name of the image registry. |
| `url` | string | Yes | The url of the image registry. |
| `username` | string | No | The username used to authenticate with the image registry. |
| `password` | string | No | The password used to authenticate with the image registry. |
| `id` | string | No | The id of the image registry. |
| `createdAt` | string (date-time) | No | The date and time when the image registry was created. |
| `updatedAt` | string (date-time) | No | The date and time when the image registry was last updated. |
