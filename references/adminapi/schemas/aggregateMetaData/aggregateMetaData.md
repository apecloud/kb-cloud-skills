# aggregateMetaData

A series of data points for meta data, including a count and a timestamp.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `total` | integer | Yes | The total count of the metaData items. |
| `items` | seriesItem[] | Yes | The list of series items aggregated in the time range. |
