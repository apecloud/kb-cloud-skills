# storageClassUpdate

StorageClassUpdate provides the fields for updating a specific storage class.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `isDefaultClass` | boolean | Yes | Indicates if this is the default storage class. |
| `displayName` | string | Yes | A user-friendly name for the storage class. |
| `description` | string | Yes | A detailed description of the storage class. |
| `enabled` | boolean | Yes | Indicates if the storage class is enabled. |
