## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `default` | object | No | default value for the parameter |
| `description` | string | No | description of the parameter |
| `type` | string | No | type of the parameter |
| `hidden` | boolean | No | whether to hide this parameter from frontend display |
