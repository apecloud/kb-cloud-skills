# createClassType

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Consists of lowercase English letters and numbers, 1-16 characters long. Used to add to the class type name. |
| `description` | string | No | The description of the class type. Optional. |
| `displayName` | string | No | The display name of the class type. Optional. |
| `displayNameEn` | string | No | The display name of the class type in English. Optional. If you switch the language to English, the display name will be displaynameEn. |
| `descriptionEn` | string | No | The description of the class type in English. Optional. If you switch the language to English, the description will be descriptionEn. |
| `sourceClassType` | string | No | Support to copy classes from the source class type. Optional. |
